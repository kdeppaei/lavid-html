# Lavid HTML

Standalone browser version of Lavid AI Code Workspace.

## Files

- `index.html`: main app
- `assets/lavid-tech-background-p14.png`: background asset

## GitHub Pages

Deploy from the repository root. The live page will be available at:

`https://<your-github-username>.github.io/<repo-name>/`

This version runs fully in the browser. Local compiler execution requires a backend or desktop wrapper.
