# Lavid HTML Site Deploy

This folder is a static website package.

Files:
- `index.html`: Lavid HTML app
- `assets/lavid-tech-background-p14.png`: website background asset

Fastest deploy:
1. Go to https://app.netlify.com/drop or https://vercel.com/drop.
2. Upload this whole folder, or upload `lavid_html_site_p14_20260627.zip`.
3. After deployment finishes, copy the live URL and send it to friends.

GitHub Pages deploy:
1. Create a GitHub repository.
2. Upload `index.html` and the `assets` folder to the repository root.
3. In repository Settings, enable Pages from the selected branch.
4. Share the GitHub Pages URL after it is published.

Note:
This HTML version runs fully in the browser. It can store data locally in each user's browser, import files, preview resources, scan references, and export files. A pure browser page cannot directly run local Python/C/C++/MiKTeX compilers on other users' devices without a backend or desktop wrapper.
